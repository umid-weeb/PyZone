# Arena React App

This folder contains the React refactor of the Pyzone Arena frontend.

## Commands

```bash
npm install
npm run dev
npm run build
```

Production build output is written to:

```text
../public/arena-spa
```

## Structure

```text
arena/
  src/
    components/
    context/
    hooks/
    lib/
    pages/
    styles/
```

## Routes

- `/zone`
- `/login`
- `/register`
- `/profile`
- `/submissions`
- `/leaderboard`
- `/settings`
